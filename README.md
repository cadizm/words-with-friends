Enumerate words that can be made from a given set of letters

Given a string of letters (includes "spaces"), print the valid English words that can be formed.

For example:

```
$ docker run words 'abcd'
2 ab
2 ad
2 ba
3 bad
3 cab
3 cad
3 dab
```
